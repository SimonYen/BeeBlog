# BeeBlog
由Beego框架完成的Blog Web应用。

为什么要叫BeeBlog呢？因为我采用的框架为Beego,本身的吉祥物就是蜜蜂，所以我就之间取名为BeeBlog.

如果您仔细观察网页背景，您会发现背景图都是花卉，正好与bee采花的意境相烘托。
